CSS can be addes to HTML documents in 3 ways:

  • Inline by using style attribute inside HTML elements
  • Internal by using style element in the head section
  • External by using a link element to link to an external CSS file
  • The most common way to add CSS, is to keep the file in external CSS files. However in this tutorial we will use inline & internal styles.because this easier to demomstrate and is easier for you to try it ypurself.

IN CSS , selectors are used to target a specific element or range of elements on a web page.Once an element has been targeted , a style or set of style can be applied to the element
